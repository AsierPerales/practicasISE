# practicasISE D1

El objetivo de estas sesiones de prácticas es familiarizarse con la instalación, modificación, monitorización y optimización de servidores virtuales.
Se utilizaron versiones ISO de Debian Linux y Alma Linux adecuadas para el trabajo y ejecutadas en VirtualBoxVM.

# Objetivos de Cada práctica

## Instalación virtualizada de SO en servidores
Saber instalar el SO más conveniente para el servidor, configurar un RAID1 o LVM encriptado (u otras opciones de hardware/software) según las necesidades del usuario, así como configurar una red local básica y una interfaz de red funcional para instalar los servicios necesarios posteriores.

## Configuración de servicios
Entender los gestores de paquetes presentes en el SO con tal de instalar nuevas aplicaciones: SSH, LAMP stack y git entre otras. Además configuramos estos servicios para poder desplegar servidores web de forma segura con LAMP; por ende, indirectamente aprenderemos herramientas de seguridad en redes como firewall-cmd, ufw, ss, etc. Fueron esenciales métodos para acceder de forma segura a nuestro servidor y evitar ataques externos con SSH sin contraseña, listas de usuarios o prevención de atques de fuerza bruta. Tambíen entendimos diferentes métodos de backup y control de versiones para dar fiabilidad extra al servidor (git y herramientas nativas como tar y rsync).
